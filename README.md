# 水利マップ (SuiriMap)

農業用水路や関連施設（水門、取水口など）の位置・系統を可視化し、現場やオフィスから直感的に編集・管理するためのWebアプリケーションです。

---

## 🌟 特徴

1. **直感的な水路・施設編集**:
   - 地図上をクリックして水路をプロット可能（ダブルクリックで確定）。
   - プロット中に間違えた場合は、`Ctrl + Z` または **地図上右クリック** で1点ずつ戻る（Undo）ことができます。
   - 地図上をクリックして、取水口や水門などの「施設ピン」を配置可能。
2. **多段階で動くスマートな起動カメラ（Googleマップ互換）**:
   - **位置記憶**: 直前に見ていた地図の位置・ズームを自動記憶し、再起動時に同じ場所から再開。
   - **データ自動フィット**: 記憶がない場合、登録されている水路データがある場所に自動でジャンプ・ズーム。
   - **GPS現在地連動**: データもない場合、ブラウザの位置情報（GPS）から現在地を中心に表示。
   - **広域フォールバック**: 上記すべてが不明な場合、日本全体（広域全図）からスタートします。
3. **データ連携とバックアップ**:
   - サーバー上のデータベース（`waterways.json`）への保存。
   - ローカルPCの任意の場所へ GeoJSON 形式で出力（別名保存・エクスポート）。
   - PC上のデータファイルを読み込み、自動的にデータの場所にカメラをジャンプさせて編集開始（インポート）。
4. **数字スピナー・太さ/色連動UI**:
   - 系統番号は「+」「-」ボタンで直感的にトグル増減。
   - 水路の太さ（本線/支線）に合わせて、選択可能な線の色が「濃い色」と「薄い色」に自動連動制限され、表記もシンプル化。

---

## 🛠️ 技術スタック

* **バックエンド**: FastAPI (Python)
* **フロントエンド**: Vanilla HTML, CSS, JavaScript
* **地図ライブラリ**: Leaflet.js
* **地図データ**: 国土地理院（標準地図・空中/航空写真タイル）

---

## 🚀 クイックスタート

### 1. 依存ライブラリのインストール
リポジトリ直下で以下を実行し、FastAPI などの必要なパッケージをインストールします。

```bash
pip install -r requirements.txt
```

### 2. ローカルサーバーの起動
Uvicorn を使用してローカルサーバーを起動します。

```bash
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

### 3. ブラウザで開く
ブラウザで以下のURLにアクセスします。
[http://127.0.0.1:8000/](http://127.0.0.1:8000/)

---

## 📂 ディレクトリ構成

* `app/`
  * `main.py` - FastAPI バックエンドロジック（データの読み書き・画像保存API）
  * `static/` - フロントエンドの静的ファイル（HTML, CSS, JS）
* `data/`
  * `waterways.json` - 登録された水路・施設を格納する GeoJSON データベース
  * `images/` - アップロードされた施設の写真フォルダ
* `requirements.txt` - Python のライブラリ依存定義
